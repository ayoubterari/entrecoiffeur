# Icônes PWA pour EntreCoiffeur

## Génération des icônes

Pour générer les icônes PWA aux bonnes dimensions, vous pouvez utiliser un outil en ligne comme :
- https://www.pwabuilder.com/imageGenerator
- https://realfavicongenerator.net/

## Dimensions requises

Les icônes suivantes doivent être créées :
- icon-72x72.png (72x72 pixels)
- icon-96x96.png (96x96 pixels)
- icon-128x128.png (128x128 pixels)
- icon-144x144.png (144x144 pixels)
- icon-152x152.png (152x152 pixels)
- icon-192x192.png (192x192 pixels)
- icon-384x384.png (384x384 pixels)
- icon-512x512.png (512x512 pixels)

## Instructions

1. Créez un logo carré de 512x512 pixels avec le logo EntreCoiffeur
2. Utilisez un des outils ci-dessus pour générer toutes les tailles
3. Placez les fichiers générés dans ce dossier `/public/icons/`
4. Les icônes seront automatiquement utilisées par le manifest.json

## Recommandations

- Utilisez un fond de couleur #C0B4A5 (couleur principale de la marque)
- Le logo doit être centré et occuper environ 70% de l'espace
- Format PNG avec transparence si possible
- Optimisez les images pour réduire la taille des fichiers
